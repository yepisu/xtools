小 Tips
轻松两步骤搞定 Nginx Basic Authentication 配置！
1. 创建文件 /etc/nginx/.htpasswd 并将上述密文写入
2. 在对应的 location 修改配置如下，并 nginx -s reload
server {
  auth_basic           "Administrator’s Area";
  auth_basic_user_file /etc/nginx/.htpasswd;
}
